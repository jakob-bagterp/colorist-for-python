# OKLCH Colors
## What are OKLCH Colors?

| Parameter      | Lightness                | Chroma                  | Hue                        |
| -------------- | :----------------------: | :---------------------: | :------------------------: |
| Allowed values | `0.0`-`1.0`              | `0.0`-`0.4`             | `0`-`360` degrees          |
| Description    | Brightness of the color. | Intensity of the color. | Degree on the color wheel. |
